### 函数参数的传递

(1)	函数调用，调用这将参数传递给函数的参数
①	实参：可以是常量变量表达式
②	形参：函数调用的参数
(2)	函数中修改形参的值不会影响实参。
(3)	代码

```c++
#include <iostream>
int max(int a, int b);
int min(int a, int b);
int main()
{
    int a = 10;
    int b = 11;
    std::cout << "max:" << max(a,b) << std::endl;
    std::cout << "Hello World!\n";
}

int max(int a, int b) {
    return a > b ? a : b;
}

int min(int a, int b) {
    return a < b ? a : b;
}
```
